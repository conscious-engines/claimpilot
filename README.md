# ClaimPilot POC

AI-powered vehicle insurance claims management agent for commercial vehicles and construction equipment.

Built for Equifi / SaleMyMachine.

## Run

Open `index.html` in a browser. No build step, no dependencies.

## Stakeholder Views

Switch between views using the dropdown in the top banner:

1. **Claimant** — WhatsApp-style chat interface with voice notes, Hinglish support
2. **Operations** — Claims dashboard with document tracking, timelines, deadline alerts, integration logs
3. **Management** — Overview metrics, fleet risk, cost summary, SLA compliance

All data is synthetic. Edit the `CLAIMS` object at the top of the script to modify.
